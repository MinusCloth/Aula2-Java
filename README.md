# Linguagem de Programação Orientada a Objetos
Estruturas de repetição
Profs. Dr. Kleython José Coriolano Cavalcanti de Lacerda / Dr. José Renato Alcarás

- 1). Escreva um algoritmo que calcule e imprima o somatório dos números pares existentes entre 100 e 500.
   
- 2). Escreva um algoritmo em JAVA que dado um conjunto com a idade de 50 pessoas digitados em sequência, informe o percentual em cada faixa etária: de 0 a 15 anos, de 16 a 30 anos, de 31 a 45 anos, de 46 a 60 anos e mais que 60 anos.
   
- 3). Escreva um algoritmo que dados os limites de um intervalo [a; b], informe a soma de todos os números naturais neste intervalo. Exemplo: [4, 7] => 4+5+6+7=22.
   
- 4). Em JAVA, escreva um algoritmo que dado um conjunto de 20 elementos numéricos digitados em sequência, informe a soma de todos os números pares.
   
- 5). Faça um programa, utilizando while, que mostre na tela de 0 até N, em que N é o limite inserido pelo usuário.
    
- 6). Escreva um programa em Java que utilize um laço while para imprimir uma contagem regressiva de 10 a 1, e então exiba "Fogo!".
    
- 7). Crie um programa em Java que solicite ao usuário que insira números inteiros positivos. Utilize um laço do-while para somar apenas os números pares inseridos, até que o usuário insira um número negativo. No final, exiba a soma dos números pares inseridos.
    
- 8). Desenvolva um programa em Java que utilize um laço while para exibir a tabuada de um número escolhido pelo usuário. O programa deve permitir que o usuário escolha qual número deseja multiplicar até que ele decida encerrar o programa.
    
- 9). Crie um jogo em Java onde o computador gera um número aleatório entre 1 e 100, e o usuário tenta adivinhar o número. Utilize um laço do-while para permitir que o usuário continue tentando até que ele adivinhe corretamente o número gerado. Se errar, informe ao usuário se o número digitado é maior ou menor do valor a ser adivinhado. Observação: Para utilizar a classe Random do JAVA, utilize: import java.util.Random; Random rand = new Random(); e o int numeroAleat=rand.nextInt(101) + 1;
    
- 10). Escreva um programa em Java que solicite ao usuário que insira uma senha. Utilize um laço while para continuar solicitando a senha até que o usuário insira a senha correta. Após inserir a senha correta, exiba uma mensagem de boas-vindas.
